# 📊 Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn using SQL, Python, and Power BI. It focuses on identifying factors that influence customer churn and provides interactive dashboards with actionable business insights.

---

## 🛠️ Tools & Technologies
- SQL
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- Excel

---

## 📂 Project Files

- 📊 Churn Analysis.pbix – Power BI Dashboard
- 🐍 Churn Prediction py.ipynb – Python Analysis & Prediction
- 🗄️ churn project sql.sql – SQL Queries
- 📄 Customer_Data.csv – Dataset

---

## 📈 Dashboard KPIs

- Total Customers
- Churn Customers
- Churn Rate
- Monthly Charges
- Tenure

---

## 📌 Key Insights

- Customers with month-to-month contracts have a higher churn rate.
- Fiber Optic customers churn more frequently.
- Long-term customers are more likely to stay.
- Higher monthly charges are associated with increased churn.
